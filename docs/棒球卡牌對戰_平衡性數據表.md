## 2. 機率模型

### 2.1 起手手牌機率

手牌 7 張，牌庫 111 張，以超幾何分佈計算。

**計算公式：**

```
P(手牌中有 k 張某種牌) = C(N,k) × C(111-N, 7-k) / C(111, 7)

其中 N = 該種牌在牌庫中的總張數
```

**關鍵起手機率：**

| 事件 | 機率 | 說明 |
|------|------|------|
| 手牌中有 ≥1 張壞球 | 70.5% | 16/111 的超幾何分佈 |
| 手牌中有 ≥1 張好球 | 58.7% | 12/111 |
| 手牌中有 ≥1 張安打 | 33.1% | 6/111 |
| 手牌中有 ≥1 張全壘打 | 18.0% | 3/111 |
| 手牌中有 ≥1 張失誤 | 33.1% | 6/111 |
| 手牌中無任何攻擊牌 | 16.2% | 27 張攻擊牌都不在手中 |
| 手牌中無任何防守牌 | 12.3% | 30 張防守牌都不在手中 |

### 2.2 球數演進機率

假設攻方隨機從壞球/好球中選擇（實際受手牌限制）：

**壞球與好球的比例：** 16:12 = 4:3

| 結果 | 預期投球數 | 說明 |
|------|-----------|------|
| 四壞保送平均投球數 | ~5.6 球 | 需 4 壞球，可能夾雜好球 |
| 三振平均投球數 | ~4.8 球 | 需 3 好球，可能夾雜壞球 |
| 打者面對平均投球數 | ~3.5 球 | 含提前打擊（球數 ≥2 出攻擊牌） |

### 2.3 攻擊牌抽到時機

**在球數 ≥2 時手中有攻擊牌的機率：**

經過 2 輪投球（已看過約 4-6 張牌），加上初始 7 張手牌，玩家約已接觸 9-13 張牌。

```
P(至少1張攻擊牌在9張中) = 1 - C(84,9)/C(111,9) ≈ 82%
```

此機率確保大多數打席能在球數門檻到達時有攻擊牌可打，避免遊戲節奏過慢。

### 2.4 全壘打觸發機率

全壘打需要：
1. 球數達到 3-3 滿球數
2. 手中持有全壘打牌

```
P(達到3-3滿球數) ≈ 8-12%（依出牌策略而異）
P(此時手中有全壘打) ≈ 25-35%（經過多輪抽牌後）
P(全壘打成功打出) ≈ 2-4% per 打席
P(全壘打未被界外球擋) ≈ 70-85%（守方可能無界外球）
P(全壘打最終生效) ≈ 1.5-3.5% per 打席
```

**設計目標：** 每場 3 局遊戲約 15-20 個打席，預期全壘打 0-1 次，偶爾 2 次。

## 3. 期望值分析

### 3.1 各攻擊牌 EV (Expected Value)

以「推進壘數 × 成功率」為基礎的簡化 EV：

| 卡牌 | 基礎推進 | 成功率 (未被反制) | 反制後推進 | 反制率 | 綜合 EV |
|------|:--------:|:----------------:|:----------:|:------:|:-------:|
| 安打 | 1 壘 | 100% (無反制) | 1 壘 | 0% | **1.00** |
| 內野滾地 | 1 壘 | ~60% | 出局(-1) | ~40% | **0.20** |
| 外野高飛 | 2 壘 | ~60% | 出局(-1) | ~40% | **0.80** |
| 觸擊短打 | 跑者+1壘, 打者出局 | ~65% | 出局(-1) | ~35% | **0.30** (戰術價值) |
| 全壘打 | 全員得分 | ~75% | 無效(0) | ~25% | **變動大** |

**EV 計算公式：**

```
EV(card) = P(success) × V(success) + P(countered) × V(countered)

V(success) = 基礎推進壘數 + 壘上跑者推進價值
V(countered) = 反制效果值 (出局 = -1, 無效 = 0)

P(countered) = P(對手手中有反制牌) × P(對手選擇出反制牌)
```

### 3.2 反應鏈期望值

| 反應深度 | 發生機率 | 說明 |
|:--------:|:--------:|------|
| 1 層 (攻擊牌生效) | ~55% | 守方無牌/不反應 |
| 2 層 (防守牌生效) | ~25% | 守方反制，攻方無失誤 |
| 3 層 (失誤逆轉) | ~12% | 攻方用失誤反制 |
| 4 層 (美技再逆轉) | ~8% | 守方用美技 |

### 3.3 每打席預期得分 (Runs per Plate Appearance)

```
E[runs/PA] = Σ P(event) × E[runs|event]

估算：
  四壞保送: 0.12 × 0.3 = 0.036
  安打:     0.25 × 0.4 = 0.100
  外野高飛: 0.15 × 0.6 = 0.090
  全壘打:   0.03 × 2.0 = 0.060
  其他攻擊: 0.10 × 0.2 = 0.020
  三振/出局: 0.35 × 0.0 = 0.000

  E[runs/PA] ≈ 0.306
```

**每場預期得分 (3 局)：** 約 15-20 打席 × 0.306 ≈ **4.6-6.1 分/隊**

## 5. 平衡性調參指南

### 5.1 可調參數

| 參數 | 目前值 | 調整範圍 | 影響 |
|------|--------|----------|------|
| 壞球張數 | 16 | 14-18 | ↑ 保送增加 → 攻方略有利 |
| 好球張數 | 12 | 10-14 | ↑ 三振增加 → 守方略有利 |
| 攻擊牌球數門檻 | 2 | 1-3 | ↓ 攻擊更快 → 遊戲節奏加快 |
| 全壘打張數 | 3 | 2-4 | ↑ 爆發力增加 → 比分波動大 |
| 全壘打球數門檻 | 3-3 | 2-2 至 3-3 | ↓ 全壘打更易觸發 |
| 失誤/美技張數 | 各 6 | 各 4-8 | ↑ 反應鏈更深 → 策略性增加 |
| 界外球張數 | 6 | 4-8 | ↑ 防守更穩 → 攻方壓力大 |
| 初始手牌 | 7 | 5-9 | ↑ 選擇更多 → 策略性增加 |
| 回收牌數 | 因牌而異 | — | 影響後半場牌庫組成 |

### 5.2 調參優先順序

如果模擬數據偏離目標：

1. **先攻勝率偏離 50%：** 調整壞球/好球比例（先攻有利→增好球/減壞球）
2. **得分過低 (<3 分/隊)：** 降低攻擊牌球數門檻 或 增加攻擊牌張數
3. **得分過高 (>8 分/隊)：** 增加防守牌張數 或 提高攻擊牌球數門檻
4. **全壘打過多 (>2 次/場)：** 減少全壘打張數 或 增加界外球張數
5. **遊戲過長 (>50 分鐘)：** 減少壞球/好球張數 或 降低初始手牌
6. **遊戲過短 (<20 分鐘)：** 增加基礎牌數量
7. **完封過多 (>15%)：** 增加攻擊牌或降低防守牌
