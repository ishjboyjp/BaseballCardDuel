## 1. Godot 本地化設定

### 1.1 TranslationServer 設定

在 `project.godot` 中：

```ini
[internationalization]
locale/translations=PackedStringArray(
    "res://localization/zh_TW.translation",
    "res://localization/en.translation",
    "res://localization/ja.translation"
)
locale/fallback="zh_TW"
```

### 1.2 CSV 格式規範

所有翻譯文本以 CSV 匯入 Godot，產生 `.translation` 資源檔。

```
keys,zh_TW,en,ja
GAME_TITLE,棒球卡牌對戰,Baseball Card Duel,ベースボールカードデュエル
```

**檔案路徑：** `localization/translations.csv`

**編碼：** UTF-8 with BOM
