## 1. 音頻風格總綱

### 1.1 整體音樂風格

**關鍵詞：** 日式棒球動畫感、熱血但輕鬆、卡牌遊戲的節奏感、下午球場的溫暖氛圍

**參考風格：**
- 棒球動畫《乒乓》《大乒乓》風格的爵士/放克配樂
- 桌遊 App 常見的輕快循環配樂
- 日本高中棒球紀錄片的陽光感

**禁止風格：** 電子舞曲 (EDM)、重金屬、恐怖氛圍、過度史詩管弦樂

### 1.2 音效風格

**設計原則：**
- 卡通化但不幼稚 — 類似體育遊戲的誇張音效
- 打擊回饋明確 — 出牌、得分的聲音要有「手感」
- 層次分明 — BGM 不搶音效，音效不互相覆蓋

### 1.3 技術規格

| 屬性 | BGM | 音效 (SFX) |
|------|-----|-----------|
| 格式 | OGG Vorbis | OGG Vorbis |
| 取樣率 | 44100 Hz | 44100 Hz |
| 聲道 | 立體聲 | 單聲道 (短音效) / 立體聲 (長音效) |
| 位元深度 | 16-bit | 16-bit |
| 響度標準 | -14 LUFS | 峰值 -3dBFS |
| 循環 | 無縫循環 | 不循環 |

### 2.3 BGM 切換邏輯

```
if game_phase == TITLE:
    play BGM-01
elif game_phase == TUTORIAL:
    play BGM-07
elif game_phase == GAME_OVER:
    play BGM-04 (win) or BGM-05 (lose)
elif current_inning > total_innings:  # 延長賽
    crossfade to BGM-06 (1.5s)
elif _is_critical_moment():
    crossfade to BGM-03 (1.0s)
else:
    play BGM-02

func _is_critical_moment() -> bool:
    return (
        ball_count.is_full_count() or
        (outs >= 2 and base_runner.has_runner(THIRD)) or
        (current_inning == total_innings and is_bottom and score_diff <= 1)
    )
```

**交叉淡入淡出：** BGM 切換時使用 1.0-1.5 秒的 crossfade，避免突兀中斷。

## 4. 音頻實作規格

### 4.1 Godot AudioBus 架構

```
Master Bus
├── BGM Bus          (Volume: -10dB, 低通濾波器在暫停時啟用)
├── SFX Bus          (Volume: -6dB)
│   ├── Card SFX     (Volume: 0dB)
│   ├── Baseball SFX (Volume: 0dB)
│   └── UI SFX       (Volume: -3dB)
└── Ambient Bus      (Volume: -12dB, 預留觀眾環境音)
```

### 4.2 AudioManager.gd

```gdscript
# src/audio/audio_manager.gd (Autoload)
extends Node

@onready var bgm_player: AudioStreamPlayer = $BGMPlayer
@onready var bgm_crossfade: AudioStreamPlayer = $BGMCrossfade

var sfx_pool: Array[AudioStreamPlayer] = []
const SFX_POOL_SIZE = 8

func _ready():
    for i in range(SFX_POOL_SIZE):
        var player = AudioStreamPlayer.new()
        player.bus = "SFX"
        add_child(player)
        sfx_pool.append(player)

func play_bgm(stream: AudioStream, fade_time: float = 1.0):
    bgm_crossfade.stream = stream
    bgm_crossfade.volume_db = -80
    bgm_crossfade.play()

    var tween = create_tween()
    tween.parallel().tween_property(bgm_player, "volume_db", -80, fade_time)
    tween.parallel().tween_property(bgm_crossfade, "volume_db", -10, fade_time)
    tween.tween_callback(_swap_bgm_players)

func play_sfx(stream: AudioStream, volume_db: float = 0.0, bus: String = "SFX"):
    for player in sfx_pool:
        if not player.playing:
            player.stream = stream
            player.volume_db = volume_db
            player.bus = bus
            player.play()
            return
    # 池滿時搶佔最舊的
    sfx_pool[0].stop()
    sfx_pool[0].stream = stream
    sfx_pool[0].volume_db = volume_db
    sfx_pool[0].play()

func _swap_bgm_players():
    var temp = bgm_player
    bgm_player = bgm_crossfade
    bgm_crossfade = temp
    bgm_crossfade.stop()
```

### 4.4 音量優先級

當多個音效同時觸發時的優先順序：

```
1. 全壘打 (SFX-19)     — 最高，壓低其他所有音效 -6dB
2. 三振/保送 (SFX-15/16) — 高，正常播放
3. 得分 (SFX-18)         — 高，正常播放
4. 棒球事件 (SFX-12~14)  — 中，正常播放
5. 卡牌互動 (SFX-01~05)  — 中低
6. UI 互動 (SFX-30~33)   — 低
```
